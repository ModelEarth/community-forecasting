Data Processing

Community datasets from 2012-2016 have been processed and consolidated into a single dataset consumable by our forecasting model. Processing the data across geographies enables the forecasting model to more robustly capture underlying patterns across locations with otherwise sparse data. Combining the annual data into a single, cohesive dataset permits a proper linear regression time series analysis.
